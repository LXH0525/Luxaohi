# ✨Luxaohi — 中文编程语言 | A Chinese Programming Language
> 中国人也许可以试试用母语编程吧?-Maybe Chinese people can try programming in their native language?

> 这主要是我一个学生和一些AI工具制作的,如有不足,还请多指教-This was mainly created by me, a student, along with some AI tools. Please feel free to offer advice if there are any shortcomings.


## ❗特性-Features
- **全中文关键字** - 如果、那么、变量、喵叫。
- **Fully Chinese keywords** - 如果(if),那么(then),变量(variable),喵叫(meowPrint).
- **开箱即用** - 单个EXE文件，无需安装。
- **Ready to use out of the box** - A single EXE file, no installation required.
- **学习友好** - 基本语法类似现有编程语言，上手可能较易。
- **Learning-friendly** - Basic syntax similar to existing programming languages, making it relatively easy to pick up.

## 🧐已知事项-Known Issues
- **环境兼容性提示**：可能出现中文显示以及命令无法正常执行问题，根据各版本的表现情况，luxaohi貌似对Windows11兼容性较好。
- **Environment Compatibility Notice**: There may be issues with Chinese character display and commands not executing properly. Depending on the version, Luxaohi seems to have better compatibility with Windows 11.
- **Microsoft Edge等浏览器安全警告**：从此仓库下载的Luxaohi解释器绝对不会包含攻击代码等有害代码，请放心下载使用。
- **Browser Security Warnings (e.g., Microsoft Edge)**: The Luxaohi interpreter downloaded from this repository absolutely does not contain any malicious code such as attack code. Please feel free to download and use it.
- **文档准确性**：Luxaohi(刘小黑)编程语言关键词的意思与用法等文档可能会出现些许不准确的情况，这通常是因为一些难以或不太容易发现的代码问题所导致的，如果您在使用过程中发现了文档与实际不相符的问题，欢迎您进行反馈，谢谢。
- **Documentation Accuracy**: There may be some inaccuracies in the documentation regarding the meaning and usage of keywords in the Luxaohi (Liu Xiaohei) programming language. This is usually caused by subtle or hard-to-find code issues. If you encounter any discrepancies between the documentation and the actual behavior, please feel free to provide feedback. Thank you.
